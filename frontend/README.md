# Placement Portal Application

## Overview

The Placement Portal Application is a web-based platform developed using Flask and Vue.js to simplify campus placement activities.

## Features

### Admin
- Login
- Dashboard
- Approve Companies
- Approve Placement Drives
- Manage Students
- Manage Companies
- View Applications

### Company
- Registration
- Login
- Company Dashboard
- Post Placement Drives
- View Applicants
- Shortlist Students
- Update Application Status

### Student
- Registration
- Login
- Student Dashboard
- View Placement Drives
- Apply for Jobs
- Track Application Status
- Upload Resume

## Technologies Used

- Flask
- SQLAlchemy
- SQLite
- Vue.js
- JWT Authentication
- Axios
